ADD-153 Assignments

  • Assignment: Learn Emmet

    Learn to use Emmet with HTML and CSS.

  • Assignment: Newsletter

    Create a simple newsletter page, use multiple heading levels and semantic divs. Test page for, and correct Accessibility Errors. View the outline created by your page. Turn in screenshots of wave results. Submit link to website.

  • Assignment: CSS Variables

    Download the example file. Unzip and open the project in VS Code. Place all of the variables at the top. Change all colors used into their own variables. Change the border-radius to a variable. Change all of the colors to your liking and the diameter of the border-radius also.

  • Assignment: Starting a Project with External CSS

    Copy the HTML and move the CSS to an external style sheet.

  • Assignment: Create a Card

    Develop a card with CSS styles such as colors, shadows, text effects, etc.

  • Assignment: Create a News Blurb

    Develop a news blurb with CSS styles.

  • Assignment: Create a Hero Image

    Create a hero image which is used to focus the user's attention to something you want them to see or go to on the website.

  • Assignment: Create an Image Gallery

    Design an image gallery using CSS styles.

  • Assignment: Putting it Together

    Design a web page using external stylesheets, box model, and most important objects to showcase that you understand each works together.

  • Assignment: FlexBox

    The object of this assignment is to get the user to use and learn flexbox and everything else involved with this great tool.

  • Assignment: Layout a Grid

    The object of this assignment is to get the user to use and learn CSS grid and how to use it in developing a layout and webpage.

  • Assignment: Design a Grocery Ad

    The object of this assignment is to get the user to use and learn CSS grid and how to use it in developing a flyer for a grocery store.

  • Assignment: Wireframe for Advanced CSS

    Create a Wireframe using the program of your choice. Submit as PDF. It is a 3 page restuarant site, you will need a wireframe for each page.

  • Assignment: @media

    Use the media tag to make a webpage that looks good in mobile, tablet and laptop modes.

  • Assignment: Move Project to BEM
